Mobile First

Consiste à développer sur la version mobile puis sur les écran plus large

Reset CSS

Les avantages :

  • Repartir sur une base neutre
  • éviter les différences d’affichage

Reset CSS

les inconvénients :

  • Perte de la sémantique offert par les navigateurs
